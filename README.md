# 🛒 Trabalho – Carrinho de Compras

## *Descrição* 
Este trabalho tem como objetivo desenvolver um **Carrinho de Compras** utilizando **HTML, CSS e JavaScript**, aplicando na prática os conceitos aprendidos em sala de aula, principalmente manipulação do DOM, eventos e organização da lógica em JavaScript.

O sistema deve simular o funcionamento de uma loja virtual, onde o usuário poderá visualizar produtos e adicioná-los a um carrinho de compras, com atualização dinâmica das informações.

## *Objetivos*
* Criar a interface de uma loja virtual
* Exibir produtos na tela
* Permitir adicionar produtos ao carrinho
* Alterar quantidade de itens
* Remover produtos do carrinho
* Calcular o valor total da compra
* Atualizar o carrinho dinamicamente com JavaScript

## *Funcionamento*
O sistema deve permitir que o usuário:

1. Visualize os produtos disponíveis
2. Clique em um produto
3. Adicione o produto ao carrinho
4. Aumente ou diminua a quantidade
5. Remova itens do carrinho
6. Veja o valor total da compra sendo atualizado automaticamente

## *Tecnologias Utilizadas*
* HTML5
* CSS3
* JavaScript
